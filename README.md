# Project 3: Around The U.S.

The Around The U.S. webpage is the third project in the Software Engineering
program at TripleTen. It was created using HTML and CSS, based on a figma design brief.

## Project features

- Semantic HTML5
- Flexbox
- Positioning
- fonts
